Sudoku Game

This folder contains a simple terminal Sudoku game.

Main file:
- file17_sudoku.py

How to run:
1. Open a terminal in the project folder.
2. Run:
   python3 sudoku_game/file17_sudoku.py

How to play:
- Choose a difficulty level: easy, hard, or advance.
- If you press Enter without choosing, the game uses easy by default.
- The board will appear with some numbers already filled in.
- Empty squares are shown with a dot (.).
- Enter the row number, column number, and then the value you want to place.
- If your answer is right, the game prints CORRECT and updates the board.
- When you finish the puzzle, the program prints a congratulations message.
- Press Ctrl + C any time to quit the game.
